## 2. 핵심 전략 — "이미 쓴 답을 재료로 쓴다"

빈칸 10개는 사실 **이미 작성한 다른 답 안에 힌트가 다 있습니다.**
새로 짜내지 말고, 아래처럼 **연결**하면 5분 만에 풀려요.

| 빈칸 | 이미 쓴 답에서 가져오기 |
|------|------------------------|
| **Q29 원칙** | Q28·Q37 "건강한 삶·정신" → *"건강(몸·정신)은 무엇과도 타협하지 않는다"* |
| **Q67 인정** | Q60 "A+ 받은 경험" · Q69 "너 덕분에 해결돼서 다행이다" → 상이 아니어도 '인정받은 순간'으로 충분 |
| **Q70 돌아간다면** | Q61 실패 경험 → *"노력만 믿지 말고 결과물까지 챙길 걸"* |
| **Q71 좋아하는 작품** | Q18 포켓몬 · Q23 게임/음악 → 좋아하는 게임 한 편을 작품처럼 적기 |
| **Q74 여행지** | Q32·Q82 "고층 아파트 야경" → *"야경이 멋진 도시(예: 홍콩·도쿄)"* |
| **Q77 닮고 싶은 인물** | Q30 "바쁘게 사는 사람" → 그런 이미지의 개발자/인물 한 명 |
| **Q86 버킷리스트** | Q21 악기·작곡 · Q80 나만의 게임 → *"내가 만든 게임 출시하기"* |
| **Q88 실패 없다면** | Q82 꿈 → *"바로 나만의 게임 스튜디오/대형 프로젝트에 도전"* |
| **Q89 강점→진로** | Q38 인내 · Q45 끈기 · Q50 문제해결 → *"버그·난제를 끝까지 붙잡는 끈기가 곧 개발자의 무기"* |
| **Q97 매력 포인트** | Q3 성실·꾸준·인내 · Q98 "맡은 일 열심히" → 그대로 매력 포인트로 |

> 💡 핵심 한 줄: **"없다"가 아니라 "이미 쓴 답을 다시 엮는다."**

## 3. 추천 순서 — 쉬운 것부터 (심리적 부담↓)

빈칸을 한 번에 다 풀려 하지 말고, **쉬운 → 어려운** 순으로.

```
1단계 (가장 쉬움, 그냥 옮겨오기)
  Q97 매력 포인트  ← Q3·Q98 복사
  Q29 원칙        ← Q28·Q37 복사
  Q89 강점→진로   ← Q38·Q45·Q50 복사

2단계 (취향, 가볍게)
  Q71 좋아하는 작품  ← 좋아하는 게임/노래
  Q74 여행지        ← 야경 도시
  Q77 닮고 싶은 인물

3단계 (꿈·상상, 재미있게)
  Q86 버킷리스트
  Q88 실패 없다면
  Q70 돌아간다면

4단계 (마무리)
  Q67 인정받은 경험
  Q33·Q72·Q78 단답 한 줄씩 보강
```

## 5. 빠르게 쓰는 3가지 팁

1. **완벽 금지** — Q34 "시작이 반이다"처럼, 일단 한 줄 쓰고 넘어가기. 완벽주의보다 채우는 게 먼저.
2. **음성으로 말하듯** — 머릿속으로 친구에게 말하듯 답하고, 그걸 그대로 타이핑.
3. **"없다"는 마지막 수단** — 정말 없으면 비워둬도 됨(질문지 규칙). 단, 위 표처럼 연결 가능한 건 꼭 한 줄.
