# 💬 대화 기록 — 방학 취미 정하기

> 날짜: 2026-06-24
> 주제: 돈 적게 들고 공부도 되는 방학 취미 찾기 → 키보드(피아노) 배우기

---

## 1. 방학 취미 아이디어 요청

**나:** 방학이 되었어. 돈이 적게 들면서 공부도 하면서 쉽고 재미있게 성장할 수 있는 취미가 뭐가 있을지 아이디어를 줄 수 있을까?

**AI 추천 정리:**

### 💻 만들고 창작하는 쪽
- **코딩 / 프로그래밍** — 무료 강의 많고, 컴퓨터만 있으면 비용 거의 없음
- **블로그 / 글쓰기** — 사고력 향상 + 포트폴리오
- **영상 편집** — 무료 툴(CapCut, DaVinci Resolve)로 시작

### 🌍 지식이 쌓이는 쪽
- **외국어 공부** — Duolingo 등 무료 앱
- **독서** — 도서관 활용하면 무료
- **온라인 무료 강의** — Coursera, K-MOOC, edX

### 🎨 몸·손을 쓰는 쪽
- **악기** — 유튜브 독학 가능 (우쿨렐레/키보드 입문 부담 적음)
- **운동 / 등산 / 러닝** — 무료, 체력·기분 향상
- **그림 / 드로잉** — 종이·연필이면 시작

---

## 2. 키보드 배우기로 결정

**나:** 악기가 좋은 거 같아. 키보드로 배우고 싶은데 값이 싸고 배우기 좋은 환경을 어떻게 만들까?

### 🎹 악기 고르기

| 종류 | 가격대 | 특징 |
|------|--------|------|
| 미디 키보드 | 3~8만원 | 자체 소리 없음 → 입문용 비추천 |
| 휴대용 키보드 | 10~20만원 | 소리 내장, 가벼움. 입문 무난 |
| 디지털 피아노 | 30만원~ | 진짜 피아노 같은 건반 무게감 |

**현실적 추천**
- **돈 아끼기** → 당근마켓/중고나라 중고 매물 (10만원 안쪽 좋은 매물 많음)
- **새 제품** → 야마하 PSR-E373, 카시오 CT-S300 (가성비 입문기)
- **건반 수** → 입문은 61건반으로 충분, 흥미 붙으면 88건반 업그레이드
- **꼭 챙길 것** → 헤드폰(밤 연습 필수), 페달은 나중에

### 📚 배우는 환경 (무료/저비용)
- **유튜브**: "피아노 독학", "비루스 피아노" 등 입문 커리큘럼
- **앱**: Simply Piano, flowkey (친 걸 인식해 피드백)
- **무료 악보**: 뮤지컬노트, 노바스코어, 'easy piano tutorial' 검색

### 🗓️ 시작 루틴
1. 하루 15~20분 — 길게보다 매일이 중요
2. 첫 2주: 도레미 위치 + 양손 따로 → 같이
3. 좋아하는 노래 한 곡 목표로 (동기부여)
4. 연습 영상 가끔 찍어 실력 변화 확인
